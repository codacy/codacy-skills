# About this project
Collection of skills to help LLMs to use the Codacy CLI to:
- Improve code reviews with Codacy data (issues, security, coverage, duplication, etc.)
- Use Codacy to improve the code (open security issues, vulnerabilities, etc.)
- Improve Codacy configuration

Read and learn about the Codacy CLI at https://github.com/codacy/codacy-cloud-cli.
