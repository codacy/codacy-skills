# Using the Codacy CLI

Read and learn about the Codacy CLI at https://github.com/codacy/codacy-cloud-cli.

Create a new skill to teach Claude how to use the Codacy CLI.

The CLI should be the single and updated source of truth for the skills. Give the basics to know how to use the CLI, but also how to use the help and CLI output to learn about commands and options you don't know and can use.
